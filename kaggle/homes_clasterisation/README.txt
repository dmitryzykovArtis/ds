Описание данных:
data.csv - набор распаршенных объявлений о продаже квартир на вторичном рынке недвижимости в одном городе.

Задача:
Нужно найти объявления о продаже одной и той же квартиры.
Идентификаторы полученных объектов (кластеров) нужно записать в отдельный столбец 'cluster' - количество уникальных значений в столбце 'cluster' определяет уникальное количество продаваемых квартир.
Пример ответа можно посмотреть в файле sample-submission.csv

В ходе решения кандидату предлагается предложить свою модель для кластеризации объявлений.

Что хочется получить:
* файл с решением (по примеру sample-submission.csv)
* код python, отражающий ход решения (можно в виде скрипта .py или в ноутбуке .ipynb)
* краткий отчет (можно в виде комментариев), содержащий описание подхода к построению модели, описание самой модели, краткие выводы и дальнейшие идеи для развития.
